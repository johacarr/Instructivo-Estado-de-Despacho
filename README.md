# Instructivo-Estado-de-Despacho
Introducción
El reporte Estado de Despacho permite ver en qué situación se encuentran las Notas de Venta que todavía no están completamente procesadas.
Sirve para saber qué pedidos aún no se entregaron, cuáles falta remitir, y cuáles falta facturar.
Responsable: Encargado de depósito y Administrativo
Ruta de acceso: [Ventas → Pedidos → Despacho → Estado de despacho]
Estados que puede tener una NV
Pendiente: La NV todavía no fue ni remitida ni facturada completamente.
Sigue en proceso.
Por Remitir: La NV ya está facturada, pero falta enviarla o terminar de enviarlo al cliente.
Por Facturar: La NV ya fue remitida, pero todavía no está facturada por completo.
Cómo está organizado el informe
Área: Cada pedido aparece dentro del área operativa a la que pertenece.
Cliente: Dentro de cada área, se listan los clientes con pedidos pendientes.
Detalle del pedido: Para cada Nota de Venta se visualiza:
Fecha
Número de NV
Producto (código y descripción)
Cantidad pedida
Cantidad remitida
Cantidad facturada
Saldos pendientes (por remitir o por facturar)
Información del cliente (nombre, zona, saldo, etc.)

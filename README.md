# Ítem Estado de Despacho - Circuito de Entrega

## Introducción
El reporte *Estado de Despacho* permite ver en qué situación se encuentran las Notas de Venta que todavía no están completamente procesadas. Sirve para saber qué pedidos aún no se entregaron, cuáles falta remitir, y cuáles falta facturar.
